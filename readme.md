# DevCamper API

> Backend API for DevCamper application ,
> which is a bootcamp directory website

## Usage

Rename the "config/cofig.env.env" to
"config/.env" and update the values/settings
to your own.

## Install dependencies

```
npm install
```

## Run app

```
# Run in dev mode
npm run dev

# Run in production mode

npm start

```
